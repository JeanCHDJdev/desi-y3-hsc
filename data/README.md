## data
-------

This directory compiles useful external data for the analysis. In `mocs`, one can find the `MOCpy` MOC files for the 4 sections of the HSC footprint used in this work. In `bias`, we compile the 3D autocorrelation measurements performed by E. Chaussidon in DESI DR2 for the galaxy bias of the spectroscopic tracer used in the context of the magnification correction in the n(z) work. Finally, in `pz`, we add the file to recover the `Phot` photometry calibrated distributions from Rau et al, 2022.
