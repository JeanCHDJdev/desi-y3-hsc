# desi-y3-hsc
-------------

Github project for storing code related to the DESI-Y3 x HSC auto and cross correlation analysis. The original HSC Rau+2022 can be found [here](https://arxiv.org/pdf/2211.16516).

This analysis made for 2 distinct papers.
One can find the cosmic shear results and code under the `cosmic-shear`, including the fiducial chains as well as reweighted chains for this analysis.

Result files can be found in the `src/statistics/results/distributions` directory.
