## data
-----------

Some of the data used for other plots of this analysis.
One can find data for the histogram n(z) densities in the `hist_density_z_*.npz` files. 
The photometric galaxy bias corrections in Appendix A can be found in the `photometric_bias_corrections.npz` file.
The cross-correlations of LRGxHSC with DR1 are in the `cross_correlation_lrgxhsc_dr1.npz` file.

In the `maps` folder, one can find the vertices used for the coverage maps presented in the footprint figure.
